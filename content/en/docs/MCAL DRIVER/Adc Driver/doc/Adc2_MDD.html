Module – ADC2

High-Level Description

The ADC2 module shall control sampling and conversion of voltages from the hardware layer into digital signals.

Known Limitations With Design

  1. INLINE functions defined in “GlobalMacro.h” are not unit tested

  2. Buffer pointer initialization could hang if adc conversion did not complete. Consider adding a timeout and possible setting of NTC 0x33 bit 1.

  3. ADC component outputs the ADC calibration compensation which is in turn used by Current Measurement to correct its offset volt . This is implementation done across two components. The measured currents from ADC and conversion from counts to volts is not done in this component as described by the FDD but current measurement component.
