Module – NHET

High-Level Description

This module implements NHET1 and HTU1 initialization per ES-34B NHET1 subfunctions, and NHET2 initialization, currently not documented in an FDD.

Design Rationale

Only one memory protection region is enabled; per TI documentation updates, HTU access to NHET memory is independent of the HTU memory protection and therefore the HTU memory protection only needs to be configured to allow access to the R4 main memory region used by the HTU.

A global variable, Nhet_Htu1RstFail_Cnt_G_lgc, is used for the HTU1 Reset Test Fail flag, because the HTU1_Init function is called from ECU Startup before the RTE is running.
